Computes the K-means algorithm in a OpenCL GPU device.

Parameters
subpopOpenCL object which contains the current subpopulation. The object is stored in global memory
selInstancesOpenCL object which contains the instances choosen as initial centroids. The object is stored in constant memory
trDataBaseOpenCL object which contains the training database. The object is stored in global memory
beginThe first individual to be evaluated
endThe 'end-1' position is the last individual to be evaluated
transposedDataBaseOpenCL object which contains the transposed training database. The object is stored in global memory
