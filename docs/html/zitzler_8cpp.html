Detailed Description

Implementation of the hypervolume metric calculation procedure.

This code is an adaptation of the original code, written in C, and owned by Eckart Zitzler. This program calculates for a given set of objective vectors the volume of the dominated space, enclosed by the nondominated points and the origin. Here, a maximization problem is assumed, for minimization or mixed optimization problem the objective vectors have to be transformed accordingly. The hypervolume metric has been proposed in:

[1] E. Zitzler and L. Thiele. Multiobjective Optimization Using Evolutionary Algorithms: A Comparative Case Study. Parallel Problem Solving from Nature - PPSN-V, September 1998, pages 292-301.

A more detailed description and extensions can be found in:

[2] E. Zitzler. Evolutionary Algorithms for Multiobjective Optimization: Methods and Applications. Swiss Federal Institute of Technology (ETH) Zurich. Shaker Verlag, Germany, ISBN 3-8265-6831-1, December 1999.
